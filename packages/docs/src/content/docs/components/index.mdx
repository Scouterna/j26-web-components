## Component Library

Welcome to the Scouterna UI component library. All components are built with web components using Stencil, making them framework-agnostic and usable in any JavaScript project.

## Available Components

### Form Components

- [**Button**](/components/button/) - A customizable button component with multiple variants

## Getting Started

### Installation

```bash
npm install @scouterna/ui-webc
```

### Usage

Import the component loader in your project:

```javascript
import { defineCustomElements } from '@scouterna/ui-webc/loader';

defineCustomElements();
```

Then use the components in your HTML:

```html
<scout-button variant="primary">Click me</scout-button>
```

## Features

- ✅ **Framework Agnostic** - Works with React, Vue, Angular, or vanilla JavaScript
- ✅ **Fully Typed** - TypeScript definitions included
- ✅ **Customizable** - Design tokens for easy theming
- ✅ **Accessible** - Built with accessibility in mind
- ✅ **Shadow DOM** - Encapsulated styles prevent conflicts

## Browser Support

All modern browsers that support Web Components:

- Chrome/Edge 67+
- Firefox 63+
- Safari 10.1+
