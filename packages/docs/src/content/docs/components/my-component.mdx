## Overview

A demo component that displays formatted names. This component takes first, middle, and last name properties and displays them in a formatted string.

## Installation

```bash
npm install @scouterna/ui-webc
```

## Usage

```typescript
// Import and register the components
import { defineCustomElements } from '@scouterna/ui-webc/loader';

defineCustomElements();
```

```html
<my-component first="Jane" middle="Marie" last="Smith"></my-component>
```

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `first` | `string` | `undefined` | The first name |
| `middle` | `string` | `undefined` | The middle name |
| `last` | `string` | `undefined` | The last name |

### Events

This component does not emit any events.
