## Overview

Displays a button that triggers an action or event, such as submitting a form or opening a dialog.

## Installation

```bash
npm install @scouterna/ui-webc
```

## Usage

```typescript
// Import and register the components
import { defineCustomElements } from '@scouterna/ui-webc/loader';

defineCustomElements();
```

```html
<scout-button variant="primary">Click me</scout-button>
```

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `"primary" \| "outlined" \| "text" \| "caution" \| "danger"` | `"outlined"` | The visual style variant of the button |
| `type` | `"button" \| "submit" \| "reset"` | `"button"` | The button type attribute |
| `icon` | `string` | `undefined` | Optional icon HTML to display |

### Events

| Event | Description |
|-------|-------------|
| `scoutClick` | Emitted when the button is clicked |
